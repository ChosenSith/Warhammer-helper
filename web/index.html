Warhammer — préparation

Salon multijoueur (optionnel), puis faction et liste — lance la partie.

Salon & équipes

Chaque joueur garde son écran habituel (ses unités, ses phases). Ici tu prépares le salon et ton équipe (1 à 10) : plusieurs joueurs peuvent partager le même numéro. La connexion temps réel (PartyKit) relaie les snapshots entre clients.

Rôle

Connectés au salon

    ✓ = a lancé la partie sur son écran. L’hôte voit qui est prêt.

    Faction

    Choisis ton armée : Seigneurs Ruinemânes (Soulblight), Peaux-Vertes (Mâchefers) ou Lames de Khorne — chaque faction a son catalogue, ses formations et ses rappels en partie.

    Le catalogue ne peut pas se charger. En ouvrant la page en double-clic (file://), le navigateur bloque souvent les modules JavaScript. Lance un petit serveur dans le dossier web puis ouvre http://localhost:8080 — par exemple sous Windows : double-clique run-server.cmd, ou dans PowerShell : cd web puis python -m http.server 8080.

    Listes enregistrées

    Enregistre la composition actuelle (faction, unités, formations, traits) dans le navigateur pour la recharger plus tard.

    Unités

    Choisis une unité dans la liste puis clique « Ajouter », ou Tout ajouter pour une entrée par type du catalogue. Tu peux aussi prendre plusieurs fois la même unité.
